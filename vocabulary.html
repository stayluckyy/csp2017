Vocabulary Page!


Sprites: A graphics object that can be moved on top of or behind a background.
Stage: a raised floor or platform, typically in a theater, on which actors, entertainers, or speakers perform.
Scripts: A set of instructions. In Scratch a script is a single stack of blocks belonging to one sprite or to the stage
Costumes: a set of clothes in a style typical of a particular country or historical period.
Camera: a device for recording visual images in the form of photographs, film, or video signals. "She faced the cameras"
Allow: give someone permission to do something.
Save: keep safe or rescue someone or something from harm or danger.
Import: indicate or signify.
Sounds: music, speech, and sound effects when recorded, used to accompany a film or video production, or broadcast.
Iteration: Repeating a step-by-step procedure several times; also used to refer to single execution of that repetitive procedure.
pair program: is an agile software development technique in which two programmers
Integrated Development Environment (IDE): Provides a developer with a way to create a program, run the program, and debug the program all within one application. work together at one workstation. One, the driver, writes code while the other, the observer or navigator, reviews each line of code as it is typed in.
Select: carefully choose as being the best or most suitable.
Delete: remove or obliterate (written or printed matter), especially by drawing a line through it or marking it with a delete sign.
Information: what is conveyed or represented by a particular arrangement or sequence of things.
Event: a thing that happens, especially one of importance.
If then structure: evaluates a Boolean expression once
Execute: To run a program or a single instruction.
Bug: Part of a program that causes an error or undesired output
Debugging: The process of figuring out why code doesn't behave as expected and eliminating bugs to make it work as desired.
Central Processing Unit (CPU, or core): The part of the computer that executes one instruction at a time.
packet: A chunk of data transmitted as a unit over a network, sometimes in smaller bursts called frames.
Router:A piece of hardware responsible for dictating the flow of packets over a network
ISP: Internet Service Providers make Internet access available to users.
Protocol: An agreed-upon method for conducting a communication.
IPv4: The Internet Protocol version 4 is the dominant protocol for routing traffic on the Internet, specifying “to” and “from” addresses using a dotted decimal such as “122.45.255.0”.
Dotted decimal notation: A representation of numerical data in which groups of base-10 numbers are separated by dots.
IPv6: The Internet Protocol version 6 provides a large number of new addresses to route Internet traffic, using “from” and “to” addresses written as colon-hexadecimal notation, such as “fe80::42:acff:feaa:1bf0”.
Domain Name System (DNS): A hierarchical system for naming resources on the Internet.
name server: A computer dedicated to providing responses to requests for Domain Name information.
